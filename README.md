# Patterns & Practices

This project is from a Udemy course on React.

It creates a simple app made to show understanding of a few React patterns and practices.

## Concepts Applied

- Standard component creation to reduce repeating code
- Hooks
  - `useState()`
  - `useRef()`
  - `useContext()`
  - Custom Hooks
- Patterns
  - Compound Components
  - Render Props
  - Debouncing
- Practices
  - Resuability
  - Configurability
